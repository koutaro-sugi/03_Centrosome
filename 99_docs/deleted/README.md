# Deleted Features ドキュメント

このディレクトリには削除・廃止された機能の記録を配置します。

## 目的
- 削除理由の記録
- 将来の参考資料
- 同じ過ちを繰り返さない

## テンプレート

```markdown
# Deleted Feature: [機能名]

## Date Deleted
YYYY-MM-DD

## Original Implementation Date
YYYY-MM-DD

## Reason for Deletion
[削除理由の詳細]

## Impact Analysis
- ユーザーへの影響: [...]
- システムへの影響: [...]
- データ移行: [...]

## Alternative Solution
[代替案・移行先]

## Lessons Learned
[学んだ教訓]

## Related Issues
- Issue #XXX
- PR #YYY
```

## ファイル命名規則
```
YYYYMMDD-deleted-feature-name.md
```
